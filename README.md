# Java Project
## SafeLog Password Validatior

A desktop GUI application developed using Java Swing that enforces strong password policies through real-time validation. It uses loops, character classification methods, and event-driven programming to check password strength and provide specific feedback. The application enhances security awareness and simulates real-world authentication systems used in enterprise environments.

# Python Project
## Resume Analyzer & Job Matcher

A web-based application built using Streamlit that analyzes resumes using NLP techniques from NLTK and similarity scoring with scikit-learn. It extracts skills, compares them with job descriptions, and generates a match score along with improvement suggestions. The system helps job seekers optimize resumes and improves their chances of getting shortlisted.

# Sql Project
## Hospital Management & Patient Analytics System

A relational database system designed using SQL to manage hospital operations and analyze patient, doctor, and treatment data. It uses normalized tables, joins, and aggregation queries to generate insights such as revenue trends, disease frequency, and doctor performance. The system demonstrates effective data management and supports decision-making in healthcare environments.
